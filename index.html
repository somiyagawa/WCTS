The Truku Seediq Language

Family: Austronesian, Atayalic branch.

Key Features

  • Phonology: 19 consonants, 4 vowels (/i, u, ə, a/). Notable dorsal consonant harmony.
  • Morphology: Rich affixation (voice/focus), productive reduplication.
  • Syntax: VOS word order, ergative alignment.

Diagram: Visual overview of Truku phonology, morphology (e.g., affix example), and syntax (e.g., VOS structure).

Fine-tuning for Low-Resource Languages

Adapting pre-trained models to a specific task/domain (like Truku) using a smaller, specialized dataset.

Challenges

Limited data, unique linguistic features not in pre-trained models.

Parameter-Efficient Fine-tuning (PEFT)

Techniques like LoRA, Prompt Tuning update only a small part of model parameters, efficient for small datasets.

For Truku Seediq

Used to develop Truku language models, TTS, and translation, showing promise with Pastor Chang's Bible translations.

Neural Machine Translation (NMT)

Translating between Truku Seediq and Mandarin/English for access, documentation, learning.

Approach

  • Transfer learning (e.g., MarianMT).
  • Data augmentation for limited parallel text.
  • Morphologically-aware modeling.

Challenges

Data scarcity (~400k+ words needed), complex morphology, evaluation.
